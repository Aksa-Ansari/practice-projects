# Bent Template Clone

This project is a simple clone of the Bent Portfolio Template made using HTML and CSS.  
It was created for practice to improve my front-end development skills.

## Features
- Clean and simple HTML structure  
- Modern CSS design  
- Sections like Intro, About, Work, Gallery, Testimonials, and Footer  
- Fully structured layout (no framework used)

## Purpose of the Project
This project helps me practice:
- Creating layouts using Flexbox  
- Styling using pure CSS  
- Understanding spacing, alignment, and structuring  
- Learning how professional templates are designed  

## Technologies Used
- HTML5
- CSS3
