# Random Quote Generator

A simple and responsive Random Quote Generator built using HTML, CSS, and JavaScript.  
It fetches inspiring quotes from a public API and displays them beautifully with author names.  
Users can also share their favorite quotes on Twitter with one click.

## Features

- Fetches random quotes from the API: [dummyjson.com/quotes](https://dummyjson.com/quotes)
- Displays quote and author dynamically
- Smooth fade-in animation for new quotes
- Fully responsive for mobile and desktop
