# Drag & Drop Board

A simple Drag & Drop project built with HTML CSS and JavaScript. Move items between columns easily.

## Features
- Drag and drop items between columns
- Beginner-friendly and simple code
- No external libraries required

## Technologies Used
- HTML
- CSS
- JavaScript

## Usage
1. Open `index.html` in your browser.
2. Drag any item and drop it into another column.

## What I Learned
- How drag and drop works in JavaScript
- Selecting multiple elements using `querySelectorAll`
- Handling events like `dragstart`, `dragend`, and `dragover`
- Moving elements dynamically using `appendChild`
- Basic DOM manipulation skills
