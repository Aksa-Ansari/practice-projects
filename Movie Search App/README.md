# Movie Search App
A simple, responsive Movie Search App built using HTML, CSS, and JavaScript. This app allows users to search for movies by name and displays detailed information fetched from the OMDb API

## Features
- Search movies by name.
- Displays movie details:
  - Title & Year
  - Genre
  - Actors
  - Plot (trimmed if too long)
  - Poster image
- Smooth fade-in and fade-out effect for results.
- Loading indicator while fetching data.
- Responsive design for desktop and mobile screens.
- Search via Enter key or button click.
- Automatic input clearing and focus after search.
- Button disabled during API fetch to prevent multiple requests.

## Technologies Used
- HTML
- CSS
- JavaScript
- OMDb API
